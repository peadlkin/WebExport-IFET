# WebExport-IFET — Контекст проекта

## О проекте
Проект WebExport-IFET — веб-приложение для экспорта данных из Telegram.

## Структура проекта

```
WebExport-IFET/
├── api/                  # API endpoints
│   └── ifet/
├── bug/                  # Страница сообщения об ошибках
├── fonts/                # Шрифты
├── icons/                # Иконки
├── support/              # Страница поддержки
├── telegram_worker/      # Telegram worker скрипты
├── uninstall/            # Страница деинсталляции
├── welcome/              # Страница приветствия
├── common.css            # Общие стили
└── lang.js               # Языковые настройки
```

## Технологии
- Vanilla JavaScript
- CSS
- Telegram Web API

## Правила разработки

### Стиль кода
- Используйте консистентное форматирование
- Комментируйте сложную логику
- Используйте осмысленные имена переменных

### Git workflow
- Работаем на ветках с префиксом `claude/`
- Делаем понятные commit messages
- Push только после проверки изменений

### Безопасность
- Не коммитить API ключи и credentials
- Проверять XSS уязвимости
- Валидировать пользовательский ввод

## Локальные заметки
Для личных заметок используйте файлы в `context/` — они не будут коммититься.
