# Context Directory

Эта директория предназначена для локальных файлов контекста.

## Что здесь хранить:
- Личные заметки о проекте
- Временные идеи и концепции
- Локальные конфигурации для разработки
- Документация, которая не готова к коммиту

## Файлы, которые не коммитятся:
- `local-notes.md` — ваши личные заметки
- `personal/` — директория для личных файлов

## Использование с Claude Code:
В файле `CLAUDE.md` в корне проекта есть ссылка `@/context/local-notes.md`,
что позволяет Claude видеть ваши локальные заметки, но они не попадут в git.
